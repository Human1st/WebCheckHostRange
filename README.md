
# WebCheckHostRange

A python3 tool to search for an specific webhost in an Ip Range


## Installation

Install WebCheckHostRange

```bash
  git clone https://github.com/Human1st/WebCheckHostRange
  cd Human1st
```

## Example

```bash
python3 -r 127.0.0.1/32 -i google.com -k Google -p 80
```
